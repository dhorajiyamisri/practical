# 📌 Project Overview

This project demonstrates a complete Data Preprocessing and Feature Engineering Pipeline using ride-sharing data collected from multiple sources including Excel, JSON, and SQL databases.

The objective is to transform raw ride data into a clean, structured, and machine-learning-ready dataset suitable for analytics, reporting, and predictive modeling.

---

# 🎯 Project Objectives

✅ Load data from multiple sources

✅ Handle missing values

✅ Remove duplicate records

✅ Detect and treat outliers

✅ Perform data transformation

✅ Apply feature scaling

✅ Create new engineered features

✅ Generate final machine-learning-ready dataset

# 🔧 Data Preprocessing Techniques

### Missing Value Handling

* Mean Imputation
* Most Frequent Imputation
* KNN Imputer

### Outlier Treatment

* Z-Score Method
* IQR Method

### Data Transformation

* Datetime Feature Extraction
* Log Transformation

### Encoding

* Label Encoding
* One Hot Encoding

### Scaling

* StandardScaler
* MinMaxScaler
